# songscraper
scrap the songs from web

<p>
<pre>
<code>
Options:
-l = is latest ?
-c = concurrency
-sp = start page
-ep = end page
-s = movie id start
-e = movie id end
</code>
Sample
<code>
python3 scrap.py -l -c 8
</code>
(Downloads songs from latest page with concurrency 8)

<code>
python3 scrap.py -sp 3 -ep 10 -c 10
</code>
(Downloads all the songs from page 3 to 10 with conurrency 10)
</pre>
